# AIAA Patch V64

Official Level 1 to Level 5 exam blueprint and question bank.

This replaces the temporary six question exam bank.

Question counts:

Level 1: 30 multiple choice questions and 1 practical task.
Level 2: 40 multiple choice questions and 2 practical tasks.
Level 3: 50 multiple choice questions, 3 practical tasks, and 1 architecture review.
Level 4: 20 company questionnaire items, 8 document review items, and 1 demo review.
Level 5: 10 Fellow impact review items. No standard multiple choice exam.

Run:

node scripts\install-official-exam-blueprint-v64.cjs

Then run the SQL file in Supabase SQL Editor:

supabase\aiaa-official-exam-blueprint-v64.sql
